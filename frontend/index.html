분석 설정

단일/배치 모든 분석에 적용됩니다. 분석 시작 전에 변경하세요.

MRC 파일은 자동 감지된 값이 우선 사용됩니다.
자동은 검출된 홀로 가장 적합한 Grid를 추천합니다.
Lacey 카본 그리드는 균형 또는 정밀 모드를 권장합니다.
정밀 모드에서만 작동 — 이 점수 미만의 홀은 자동 제외(0 = 비활성).
분석 모드 설명
  • 빠름 — 단일 Otsu 이진화 + 컨투어. 균일한 이미지·일괄 처리에 적합.
  • 균형 — Sauvola 적응 이진화 + Watershed 홀 분리. 불균일 조명·인접 홀에 강건.
  • 정밀 — 균형 + Hough Circle 앙상블 + AI 신뢰도 점수(0~1). 누락 보정·오염 자동 제거.
Grid Type 기준 요약
  • Type A · 막단백질 — 직경 0.5–5.0 µm, 원형도 ≥ 0.60
  • Type B · 단백질 복합체 — 직경 1.0–10.0 µm, 원형도 ≥ 0.50
  • Type C · 나노소재/바이러스 — 직경 0.2–2.0 µm, 원형도 ≥ 0.55
  • Type D · 대형 시편 — 직경 5.0–30.0 µm, 원형도 ≥ 0.40

사용 가이드

기본 사용법

  1. 설정 탭에서 pixel scale을 입력합니다 (MRC는 자동).
  2. 단일 파일 또는 배치 모드 탭에서 파일을 업로드합니다.
  3. 분석 시작 버튼을 누르고 진행 표시를 확인합니다.
  4. 결과 페이지에서 PDF/CSV 다운로드가 가능합니다.

QC 판정 기준

  • PASS 4개 검사(직경·원형도·밀도·커버리지) 모두 통과
  • WARN 2~3개 통과
  • FAIL 1개 이하 통과

문제 해결

  • "홀이 검출되지 않았습니다" — 픽셀 스케일을 확인하세요. nm/px이 너무 크거나 작으면 면적 필터에서 모두 걸러집니다.
  • 홀이 너무 많이/적게 검출됨 — 이미지 콘트라스트 또는 픽셀 스케일을 조정해 보세요.
  • 분석이 끝나지 않음 — 우측 상단 API 문서(/api/docs)에서 서버 상태를 확인하세요.
